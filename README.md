🚢 Titanic — Survival Prediction | EDA & Machine Learning

📌 Project Overview

The sinking of the Titanic is one of the most infamous shipwrecks in history. This project uses passenger data (age, sex, ticket class, etc.) to build a machine learning model that predicts who survived and who didn't.

This project covers:
- ✅ Exploratory Data Analysis (EDA)
- ✅ Data Cleaning & Feature Engineering
- ✅ Multiple ML Models with Evaluation
- ✅ Hyperparameter Tuning
- ✅ Final Model Summary

📂 Dataset

| Detail | Info |
|--------|------|
| **Source** | [Kaggle — Titanic Dataset](https://www.kaggle.com/datasets/yasserh/titanic-dataset) |
| **Rows** | 891 passengers |
| **Target** | `Survived` (0 = No, 1 = Yes) |
| **Format** | CSV |

📋 Features

| Column | Description |
|--------|-------------|
| `PassengerId` | Unique passenger ID |
| `Survived` | Target — 0 = No, 1 = Yes |
| `Pclass` | Ticket class (1st, 2nd, 3rd) |
| `Name` | Passenger name |
| `Sex` | Gender |
| `Age` | Age in years |
| `SibSp` | No. of siblings/spouses aboard |
| `Parch` | No. of parents/children aboard |
| `Ticket` | Ticket number |
| `Fare` | Passenger fare |
| `Cabin` | Cabin number |
| `Embarked` | Port of embarkation (C, Q, S) |
